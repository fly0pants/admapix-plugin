# AdMapix Plugin for OpenClaw

🎯 Ad intelligence & app analytics assistant powered by [AdMapix](https://www.admapix.com).

## Features

- **Ad Creative Search** — Search ad creatives by keyword, app, or advertiser
- **App Analysis** — Analyze any app's advertising strategy, distribution, and performance
- **Rankings** — App store rankings, promotion rankings, download charts
- **Download & Revenue** — Track download and revenue trends with country breakdowns
- **Market Analysis** — Industry trends, top advertisers, market share
- **Deep Research** — AI-powered deep analysis reports for complex queries
- **Competitive Intelligence** — Compare apps and identify competitors

## Installation

```bash
openclaw plugins install clawhub:admapix
```

## Setup

1. Sign up at [admapix.com](https://www.admapix.com)
2. Create an API key in your dashboard
3. Configure the plugin with your API key

## Usage

Just ask naturally:

- "搜一下 Temu 的广告素材"
- "分析抖音最近 30 天的投放策略"
- "对比 Temu 和 Shein 的广告投放"
- "Analyze TikTok's ad strategy in Southeast Asia"
- "Top 10 game advertisers this month"
